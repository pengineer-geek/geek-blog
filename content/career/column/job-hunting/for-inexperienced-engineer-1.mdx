## 未経験エンジニア転職とは？
- **これは未経験じゃない**  
  - 学生→エンジニア : これはただの新卒採用
  - エンジニア→エンジニア : 職種継続
  - エンジニア→一時的に非エンジニア→エンジニア : ブランク扱い

- **これは未経験**  
  - 営業など非エンジニア→エンジニア : 最もわかりやすいケース
  - 情報・理工系専攻→非エンジニア→エンジニア : ←ペンジニアのケース  
  - (職歴なし)フリーター→エンジニア : そもそも全て未経験扱い

- **グレーゾーン**  
  - 個人開発経験ありの非エンジニア  
    - 学習レベルなら未経験扱い : 金が稼げてるかどうか?による
    - ユーザーがつくプロダクトを作っていれば経験者扱いの可能性も

### 補足
- 情報・理工系専攻→非エンジニア→エンジニア
  - いくら学生時代に学んでいたとしても、正社員or業務委託でエンジニア業務がないと「未経験扱い」です。
  - 「情報・理工系」のバックボーンがない人と比較すると **微有利** 程度で大したアドバンテージにはなりません。

- ユーザーがつくプロダクトを作っていれば...
  - このレベルまで技術力とビジネススキルがあれば、「無理に正社員にならなくても良いんじゃないかな」というのが個人的な見解です。
  - 別で生計を立てつつ、個人開発を継続した方がスキル・年収ともに良い気がします。

## ゴール設定
未経験エンジニア転職における「ゴール」とは、
いきなり有名自社開発企業に入ることではありません。  


まず現実的な到達点は

 **受託開発・SES企業（規模100名以下）に正社員で入社すること** です。

- **会社規模**：〜100名程度の中小企業  
- **雇用形態**：正社員（契約社員や派遣からのスタートもあり得る）  
- **想定年収**：240〜300万円前後

ここを出発点にして、数年の経験を積みながらステップアップしていくのが一般的です。  

「未経験から年収600万の自社開発エンジニア！」というSNSのストーリーは、ほとんどが例外的ケース。  


現実はまず **スタートラインに立つ**こと 自体がゴールなんです。

---

## 難易度について
前提、未経験エンジニア転職の難易度は非常に高いです。

SNSや各種メディアで言われているような、

「未経験からエンジニアで〇〇万円!」
「エンジニアは人材不足だから狙い目!!」

というのは、あまりアテにしないでください。

繰り返しますが、 **難易度は高いです**

とはいえ、人によって難易度に差があります。

未経験エンジニア転職の難易度は
「年齢・学歴・職歴・開発経験・資格・学習」 etc. 
様々な要素で決まります。  

以下にチェックリストを用意してみたので、
まずはご自身のスコアを出してみましょう。

### 未経験エンジニア転職 難易度診断

- **年齢**
  - 〜24歳 … 5点
  - 25歳 … 4点
  - 26歳 … 3点
  - 27歳 … 2点
  - 28〜29歳 … 1点
  - 30歳〜 … 0点

- **職歴**
  - 業界・職種
    - IT・Web系 … 3点
    - デスクワーク系（営業・事務など） … 2点
    - その他（接客・販売など） … 1点
    - 職歴なし … 0点
  - 会社規模
    - 東証プライム規模 … 2点
    - 東証スタンダード／グロース規模 … 2点
    - 一般に名が知られた未上場 … 1〜2点
    - 小規模・零細 … 0点

- **学歴**
  - 最終学歴
    - 大学院(文系除く) … 3点
    - 大学／高専 … 2点
    - 専門（情報系） … 1点
    - それ以外（高卒含む） … 0点
  - 専攻
    - 情報系 … 2点
    - 理工系（情報以外） … 1点
    - それ以外 … 0点
  - 学校ランク
    - 有名大学 … 1点
    - それ以外 … 0点

※ 有名大学はMARCH以上が目安

- **資格・学習歴**
  - 資格
    - 応用情報以上 … 3点
    - 基本情報 … 2点
    - ITパスポート … 1点

※ その他IT資格は難易度に応じて
  - スキル
    - ポートフォリオあり … 0〜5点（クオリティ次第）
    - スクール卒 … 1点（※正直そこまで評価されない）
    - 自己学習 … 0〜3点（質疑応答で理解を証明できるか）

#### 合計スコア例
参考まで、私が未経験エンジニア転職をした際のスコアは15でした。
結果は30~40社応募して、内定は1社でした。

## 書類通過難易度（スコア別）
ここでは、最初にして最大の関門──
「書類選考」の通過難度をまとめます。

(面接以降で落選した場合は大抵、 **未経験であること** 以外の要素が原因になっているからです。)

- **20点以上**  
  受託SESなら **50%程度の確率** で書類通過。  
  条件や規模次第では自社開発企業も視野に入れられる。  

- **15〜19点**  
  受託SESなら **30%程度の確率** で書類通過。  
  自社開発は正直すでに厳しい。  

- **10〜14点**  
  受託SESの書類通過率は **15〜20%程度**。  
  （契約社員や派遣で応募すれば30%まで上がる可能性あり）  
  自社開発はまず通らない。  

- **〜9点**  
  正社員での未経験エンジニア転職は諦めることを推奨。  

---

## 「~9点」 : 年齢による補足
- **27歳以下の場合**  
  アルバイトや契約社員からキャリアを積み、正社員を目指せばまだ間に合います。

- **28歳以上の場合**  
  このタイミングで未経験エンジニア転職を目指すと、不幸になる可能性が無視できません。
  どうしてもITエンジニアになりたいなら、  
  開発以外の仕事で生計を立てつつ、独学と個人開発でサービスをリリースするルートをオススメします。
